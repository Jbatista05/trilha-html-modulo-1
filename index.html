Surgimento do HTML

(voltar)

O HTML teve seu início no início da década de 1990, com o desenvolvimento da World Wide Web por Tim Berners-Lee no CERN (Organização Europeia para a Pesquisa Nuclear).

Berners-Lee é considerado o pai da Web, e ele criou a primeira versão do HTML juntamente com o primeiro navegador web chamado "WorldWideWeb" (mais tarde renomeado para Nexus).

O que é o HTML ?

(voltar)

HTML é a linguagem de marcação padrão utilizada para criar e estruturar o conteúdo de páginas da web.

Ele fornece um conjunto de elementos e tags que definem a semântica e a estrutura do conteúdo, como texto, imagens, vídeos, links e outros elementos que compõem uma página da web.

O HTML é uma linguagem baseada em marcação, o que significa que utiliza tags para envolver e identificar diferentes partes do conteúdo.

O que são tags ?

(voltar)

As tags em HTML são elementos que permitem marcar e estruturar o conteúdo de uma página web.

Elas são compostas por um nome delimitado por sinais de menor que (<) e maior que (>), e podem conter atributos que fornecem informações adicionais sobre o elemento.

Listagem de algumas tag aprendidas durante o curso:

  • < h1 > ... < h6 > Cabeçalhos de níveis 1 a 6.
  • < p > Parágrafo
  • < a > Hiperlink
  • < ol > Lista Ordenada
  • < ul > Lista não ordenada
  • < li > Itens da lista

Atributos de uma tag

(voltar)

As tags HTML podem receber atributos que fornecem informações adicionais sobre o elemento.

Por exemplo, a tag <a> possui o atributo href para especificar o destino ou referência do link.
